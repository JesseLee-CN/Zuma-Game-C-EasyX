# Zuma-Game-C-EasyX 
基于C语言与EasyX图形库的祖玛游戏实现（南京大学数据结构与算法课程大作业）

## ✨ 项目概述
基于经典祖玛游戏的简化版实现，使用C语言和EasyX图形库开发，重点演示数据结构与算法在游戏开发中的实际应用。

## 🚀 功能特性
- **核心玩法**  
  ⚡ 彩色球链动态消除  
  🔫 鼠标控制炮台发射  
  💥 三连消判定与连锁消除（未实现）
- **游戏机制**  
  🎯 自带瞄准线  
  💯 计分系统（未实现）
- **技术实现**  
  🧮 链表实现动态球链存储  
  🎨 EasyX图形界面实时渲染  
  🔊 基础音效反馈系统（未实现）
